Handbook chapters

This handbook documents every layer of the application — the Python analysis backend, the React user interface, the Rust native-audio core, the Pioneer USB-export binary formats, the security model, the build pipeline, and the road ahead. It is written to be read cover-to-cover or used as a reference. Every chapter is richly illustrated with diagrams.

How to read this handbook

Use the sidebar to jump between chapters. Diagrams render live in your browser. Newcomers should read Part I and Part II first; implementers can jump straight to the backend, frontend, or native chapters; reviewers will want Part VI on quality and security.
