# DJ & Musiktheorie — Obsidian Vault

Standalone-Wissensvault rund ums Auflegen und die Musiktheorie dahinter. Schwerpunkt: der Zusammenhang von **BPM, Pitch und Tonart** sowie **harmonisches Mixing**.

## Öffnen

1. [Obsidian](https://obsidian.md) installieren.
2. *Open folder as vault* → diesen Ordner (`obsidian-dj-vault/`) wählen.
3. Mit [[Home]] (`Home.md`) starten.

Funktioniert auch ohne Obsidian: alles sind reine Markdown-Dateien. Die `[[Wikilinks]]` werden nur in Obsidian (oder kompatiblen Editoren) klickbar gerendert.

## Aufbau

| Ordner | Inhalt |
|---|---|
| `01 Musiktheorie` | Grundlagen: Tonarten, Intervalle, Quintenzirkel, Dur/Moll, Akkorde |
| `02 Harmonisches Mixing` | Camelot Wheel, Open Key, Key-Mixing-Regeln, Energy/Mood |
| `03 BPM und Pitch` | Pitch-Fader ↔ Tonhöhe, Key Lock, Semitone-Tabellen, Genre-BPM |
| `04 DJ-Technik` | Beatmatching, EQ, Phrasing, Transitions, Looping, Cues, FX |
| `05 Tricks und Cheatsheets` | Verdichtete Spickzettel + Quick Reference |
| `06 Hardware und Software` | CDJ-3000, Pitch-Ranges, Rekordbox/Key-Analyse |
| `_meta` | Tag-Index + Notiz-Template |

## Konventionen

- **Sprache:** Deutsch; etablierte DJ-/Theorie-Fachbegriffe bleiben englisch (Key Lock, Camelot, Phrasing …).
- **Verlinkung:** jede Notiz hat oben `tags:` im Frontmatter und unten einen `## Siehe auch`-Block.
- **MOCs** (Maps of Content) sind die Einstiegs-/Hub-Notizen pro Themenbereich.
- Neue Notiz? Vorlage unter `_meta/Note Template.md`.
