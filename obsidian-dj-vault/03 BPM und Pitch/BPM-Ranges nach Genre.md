# BPM-Ranges nach Genre

Typische Tempo-Bereiche. Hilft, **mixbare** Tracks zu finden (kleine BPM-Differenz = wenig Pitch = wenig [[BPM-Sprung und Key-Änderung|Key-Drift]]).

| Genre | BPM (typisch) |
|---|---|
| HipHop / Trap (half) | 70–100 |
| Downtempo / Lo-Fi | 70–100 |
| Disco | 110–125 |
| House | 120–128 |
| Tech House | 122–128 |
| Deep House | 118–124 |
| Techno | 125–145 (Peak 130–140) |
| Trance | 130–140 |
| Hardtechno / Hardgroove | 145–160 |
| Drum & Bass | 160–180 (meist 172–176) |
| Dubstep | 138–142 (half-time-Feel ~70) |
| Hardstyle | 150–160 |

## Mix-Faustregeln

- **Innerhalb ±3–4 % BPM** mixen → minimaler Pitch, Key bleibt praktisch erhalten ([[Semitone-Pitch-Tabelle]]).
- Größere Sprünge → [[Master Tempo (Key Lock)|Key Lock]] oder [[Double und Half Time Mixing]].
- Genre-Übergänge laufen oft über **half/double time** (DnB 174 ↔ HipHop 87).

> [!tip] Set-Tempo planen
> Viele DJs starten am unteren Rand des Ziel-Genres und steigern über das Set um ein paar BPM ([[Stimmung und Energie|Energie-Bogen]]). +5–8 BPM über eine Stunde fällt kaum auf, hebt aber spürbar die Energie.
