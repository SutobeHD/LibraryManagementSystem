# Hot Cues

Gespeicherte **Sprungmarken** im Track. Per Knopfdruck springst du sofort an die Stelle — fürs Einsteigen, Re-Triggern, Mashups.

## Typische Cue-Belegung

| Cue | Position |
|---|---|
| A | Track-Start / erster Beat |
| B | Drop / Main-Part |
| C | Breakdown |
| D | Outro-Beginn (für [[Transitions|Loop-Out]]) |

## Nutzen

- **Schneller Einstieg** an der Phrasengrenze ([[Phrasing und Songstruktur]]) statt langes Suchen.
- **Re-Trigger** für Stutter/Build-Effekte.
- **Mashup:** Acapella-Cue über instrumentalen Track legen (Achtung [[Key-Mixing-Regeln|Key]]!).

> [!tip] Cues auf Phrasenstarts setzen
> Setze Hot Cues immer auf den **„1"-Beat** einer Phrase. Dann landet jeder Sprung sauber im Raster und Übergänge bleiben phrasengenau.

## Software

Rekordbox/Serato speichern Cues dauerhaft im Track-Tag bzw. der Library. Beim Export auf USB ([[Rekordbox und Key-Analyse]]) wandern sie mit.
