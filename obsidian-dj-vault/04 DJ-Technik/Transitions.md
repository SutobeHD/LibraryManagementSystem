# Transitions

Die konkreten **Übergangs-Techniken** von Track A nach B. Bauen auf [[Beatmatching]] + [[Phrasing und Songstruktur]] auf.

## Techniken-Übersicht

| Technik | Wie | Wann |
|---|---|---|
| **Blend / Long Mix** | beide laufen, langsam crossfaden + [[EQ Mixing|EQ-Swap]] | House/Techno, harmonisch |
| **Cut / Slam** | auf der „1" hart umschalten | Drops, HipHop, Energie |
| **EQ-Transition** | reiner [[EQ Mixing|Bass-Swap]], kein Crossfader | sauberster Club-Standard |
| **Echo-Out** | A mit [[Effekte (FX)|Delay/Reverb]] auflösen, B rein | Breakdown-Einstieg |
| **Filter-Fade** | A per Hi-/Lowpass wegfiltern | stimmungsvoll, verzeiht Key-Drift |
| **Loop-Out** | A in [[Looping|Loop]] halten, B aufbauen | Zeit gewinnen |

## Standard-Blend (Schritt für Schritt)

1. B beatmatchen + auf Phrasengrenze cuen.
2. B mit gekilltem Low einblenden ([[EQ Mixing]]).
3. Hats/Highs zuerst mischen.
4. An der „1": **Bass-Swap**.
5. A-Mids/Highs rausnehmen, A komplett raus.

> [!tip] Key + Transition zusammen denken
> Passt der [[Key-Mixing-Regeln|Key]] perfekt → langer Blend möglich (Melodien dürfen überlappen). Passt er nur grob → kurzer Übergang über tonal leere [[Phrasing und Songstruktur|Drum-Parts]] oder [[Effekte (FX)|Filter]].
