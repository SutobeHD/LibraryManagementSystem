# CDJ-3000

Pioneer-Standard-Player im Club. Relevante Funktionen für die Theorie dieses Vaults.

## Wichtige Bedienelemente (theorie-relevant)

| Element | Funktion | Notiz |
|---|---|---|
| **Tempo-Fader** | Pitch in % | ändert Tonhöhe → [[Pitch Fader und Tonhöhe]] |
| **Tempo-Range** | ±6 / ±10 / ±16 / WIDE | [[Tempo-Range und Pitch-Bereich]] |
| **Master Tempo** | Key Lock an/aus | [[Master Tempo (Key Lock)]] |
| **Key-Display** | Tonart-Anzeige | aus [[Rekordbox und Key-Analyse|Rekordbox]]-Analyse |
| **Key Sync / Shift** | Tonart anpassen | hebt/senkt Key in Halbtönen |
| **Hot Cues / Loops** | Marken | [[Hot Cues]] / [[Looping]] |

## Key-Features

- **Key-Display + Key Sync:** zeigt die Tonart und kann zwei Decks **harmonisch** angleichen — der Player pitcht dann gezielt um Halbtöne ([[BPM-Sprung und Key-Änderung]]).
- **Key Shift:** Track in Halbton-Schritten transponieren, **ohne** Tempo zu ändern (digitaler Pitch nur auf die Tonhöhe).

> [!info] Default-Range ±6 %
> Werkseinstellung ist oft **±6 %** — praktisch genau **±1 [[Halbton und Ganzton|Halbton]]** am Anschlag. Für größere Tempo-Anpassungen Range auf ±10/±16 stellen.

## Bezug zu diesem Projekt

Dieser Vault liegt im Repo des **Music Library Manager**, der USB-Sticks exportiert, die CDJ-3000 nativ lesen ([[Rekordbox und Key-Analyse]]).
